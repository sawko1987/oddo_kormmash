# Инструкция по установке и поиску модуля MRP XLS Import

## Проблема: Модуль не виден в меню

Если вы не можете найти модуль "Import from XLS" в меню Manufacturing, выполните следующие шаги:

## Шаг 1: Установка модуля

1. **Убедитесь, что Odoo сервер запущен**
   ```bash
   python odoo-bin -c odoo.conf
   ```

2. **Войдите в Odoo как администратор**

3. **Перейдите в раздел Apps (Приложения)**
   - Нажмите на кнопку "Главное меню" (иконка меню в левом верхнем углу)
   - Выберите "Приложения" (Apps)

4. **Найдите модуль MRP XLS Import**
   - В поиске введите: `MRP XLS Import` или `mrp_xls_import`
   - Или снимите фильтр "Apps" и выберите "All" или "Custom"
   - Найдите модуль в списке

5. **Установите модуль**
   - Нажмите на карточку модуля
   - Нажмите кнопку "Install" (Установить)
   - Дождитесь завершения установки

## Шаг 2: Поиск меню после установки

После установки модуля меню должно появиться в двух местах:

### Вариант 1: Manufacturing → Products (Товары)
1. Откройте главное меню
2. Выберите "Производство" (Manufacturing)
3. Нажмите на кнопку "Товары" (Products) в верхнем меню
4. В выпадающем меню найдите "Import from XLS"

### Вариант 2: Manufacturing → Configuration (Конфигурация) - только для менеджеров
1. Откройте главное меню
2. Выберите "Производство" (Manufacturing)
3. Нажмите на кнопку "Конфигурация" (Configuration) в верхнем меню
4. В выпадающем меню найдите "Import from XLS"

## Шаг 3: Если меню все еще не видно

### Проверка 1: Обновите модуль
```bash
python odoo-bin -c odoo.conf -d odoo -u mrp_xls_import --stop-after-init
```

Затем перезапустите Odoo:
```bash
python odoo-bin -c odoo.conf
```

### Проверка 2: Обновите страницу браузера
- Нажмите Ctrl+F5 для полной перезагрузки страницы
- Или очистите кэш браузера

### Проверка 3: Проверьте права доступа
- Убедитесь, что у пользователя есть права на модуль Manufacturing
- Пользователь должен быть в группе "Manufacturing / User" или "Manufacturing / Manager"

### Проверка 4: Проверьте логи Odoo
Если модуль не устанавливается, проверьте логи на наличие ошибок:
- Файл логов обычно находится в `var/log/odoo.log` или выводится в консоль

## Шаг 4: Прямой доступ к wizard (если меню не работает)

Если меню не отображается, можно попробовать открыть wizard напрямую через URL:

```
http://localhost:8069/web#action=mrp_xls_import.action_mrp_xls_import_wizard&model=mrp.xls.import.wizard&view_type=form
```

Замените `localhost:8069` на адрес вашего Odoo сервера.

## Проверка структуры модуля

Убедитесь, что модуль находится в правильной директории:
```
c:\oddo_kormmash\addons\mrp_xls_import\
```

И содержит следующие файлы:
- `__init__.py`
- `__manifest__.py`
- `models/mrp_bom_import.py`
- `wizard/mrp_xls_import_wizard.py`
- `views/mrp_xls_import_views.xml`
- `security/ir.model.access.csv`

## Если ничего не помогает

1. Проверьте, что модуль Manufacturing установлен и работает
2. Убедитесь, что путь к модулям в `odoo.conf` включает директорию `addons`
3. Проверьте логи Odoo на наличие ошибок при загрузке модуля
4. Попробуйте переустановить модуль: сначала "Uninstall", затем "Install"
