# MRP Responsible Assignment

Модуль для распределения ответственности в производстве.

## Установка

1. Убедитесь, что модуль находится в директории `addons/`
2. Обновите список приложений в Odoo
3. Найдите модуль "MRP Responsible Assignment" и установите его

## Функциональность

- Автоматическое назначение ответственных мастеров на операции
- Мобильные представления для мастеров
- Отслеживание причин невыполнения
- Уведомления о новых заданиях
- Отчеты по причинам невыполнения
