# Настройка отслеживания материалов

## Типы отслеживания

### 1. Отслеживание по серийным номерам (Serial Numbers)

**Применение:**
- Критичные комплектующие
- Дорогостоящие компоненты
- Оборудование с гарантией
- Компоненты, требующие индивидуального учета

**Примеры продуктов:**
- Электродвигатели
- Гидронасосы
- Контроллеры
- Датчики

**Настройка:**
1. Откройте карточку продукта
2. Перейдите на вкладку **Inventory**
3. В поле **Tracking** выберите **By Unique Serial Number**
4. Сохраните

**При приемке:**
- Система требует указания уникального серийного номера для каждой единицы
- Можно сканировать штрих-коды серийных номеров
- Каждый номер отслеживается индивидуально

### 2. Отслеживание по партиям (Lots)

**Применение:**
- Материалы с ограниченным сроком годности
- Материалы, требующие контроля партии
- Сырье с сертификатами качества на партию

**Примеры продуктов:**
- Смазочные материалы
- Краски и покрытия
- Химические реагенты
- Металл с сертификатом на партию

**Настройка:**
1. Откройте карточку продукта
2. Перейдите на вкладку **Inventory**
3. В поле **Tracking** выберите **By Lots**
4. Сохраните

**При приемке:**
- Система требует указания номера партии
- Можно указать срок годности партии
- Все единицы партии имеют одинаковые характеристики

### 3. Без отслеживания (No Tracking)

**Применение:**
- Стандартные материалы
- Массовые товары
- Материалы без индивидуальных характеристик

**Примеры продуктов:**
- Металлопрокат (листы, прутки)
- Крепеж (болты, гайки, шайбы)
- Провода и кабели
- Сыпучие материалы

**Настройка:**
1. Откройте карточку продукта
2. Перейдите на вкладку **Inventory**
3. В поле **Tracking** выберите **No Tracking**
4. Сохраните

**При приемке:**
- Учитывается только количество
- Нет необходимости указывать серийные номера или партии

## Массовая настройка отслеживания

### Через интерфейс

1. Перейдите в **Inventory → Products**
2. Используйте фильтры для выбора продуктов
3. Выберите несколько продуктов
4. Нажмите **Action → Set Tracking**
5. Выберите тип отслеживания
6. Подтвердите

### Через импорт

Создайте CSV файл с колонками:
- `name` или `default_code` - название или код продукта
- `tracking` - тип отслеживания: `none`, `lot`, `serial`

Пример:
```csv
default_code,tracking
MOTOR-001,serial
BEARING-001,lot
BOLT-M8,none
```

Импортируйте через **Inventory → Products → Import**

## Примеры конфигурации для машиностроения

### Критичные компоненты (Serial Numbers)
- Гидроцилиндры
- Электродвигатели
- Редукторы
- Контроллеры ПЛК

### Материалы с сертификатами (Lots)
- Сталь конструкционная (сертификат на партию)
- Сварочные материалы
- Покрытия и краски
- Смазочные материалы

### Стандартные материалы (No Tracking)
- Металлопрокат
- Крепеж
- Провода
- Шланги и фитинги

## Проверка отслеживания

1. Создайте заказ на закупку с продуктом, имеющим отслеживание
2. При приемке товара на склад система запросит:
   - Серийные номера (для serial)
   - Номера партий (для lot)
3. Проверьте историю перемещений через **Inventory → Reporting → Traceability**

## Отчеты по отслеживанию

- **Inventory → Reporting → Traceability Report**: полная трассировка продукта
- **Inventory → Reporting → Lot/Serial Number**: отчет по партиям и серийным номерам
- **Inventory → Products → [Product] → Traceability**: трассировка конкретного продукта
